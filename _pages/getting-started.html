As a newbie to Raspberry Pi, You may already have many questions. After reading this article, you'll find some answers to that.

The Raspberry Pi is a low-cost, credit-card-sized computer also capable of doing almost everything we do on a regular desktop computer.

Let's first look at what Raspberry Pi you should buy and how to set up the Pi.

Since you are new to Raspberry Pi, let's take the easiest way to set it up. For other methods of setting up a Pi, look for the other tutorials on this site.

One additional advantage of Raspberry Pies is that you can experiment with them as you want. You can try anything without risking the cost of hardware as experimenting with pcs.

What Raspberry Pi to buy?

රාස්බෙරි පයි වර්ග ගොඩක් තියෙනවා. ඒත් ගොඩක් අය තෝරගන්නෙ රාස්බෙරි පයි 4 model B. ඒක තමා අලුත්ම, වේගවත්ම හා වඩාත් පහසුවෙන්ම භාවිතා කරන්න පුලුවන් රාස්බෙරි පයි එක.එකෙත් ram එක 2GB එක ගොඩක් වැඩ කරගන්න ඇති.

 

මේ තියෙන්නෙ රාස්බෙරි පයි 4එකක්. image reference: ranspberrypi.org

මීට වඩා පොඩි රාස්බෙරි පයි වර්ගත් තියෙනවා. රාස්බෙරි පයි zero කියන්නේ එවගෙ එකක්. ඒවට current එක යනව අඩු නිසා රොබෝල හදන්න වගේ ගැලපෙනවා.

ඒත් රාස්බෙරි පයි 4 එක පාවිච්චි කරන්න ලේසි නිසා ඒකෙන් මුලින් prototype එකක් හදල පස්සේ ඔන්නම් පොඩි එක්කට මාරු වෙන එක ලේසියි.

මොනවද අවශ්‍ය කරන අනිත් දේවල්

  • micro SD card එකක්
    රාස්බෙරි පයිවල files සහ OS(operating system) එක කියන දෙකම save කරගන්න hard drive එක වගේ වැඩ කරන්නේ මේ SD card එක.

අඩුම තරමේ 8GB SD card එකක්වත් ඕන කරනව. රාස්බෙරි පයිවලටම setup කරපු SD card උනත් ගන්න තියෙනවා.

  • Key board and mouse
    රාස්බෙරි පයි එක set up කරගන්න USB keyboard එකක් ඕන වෙන්ව.
    Set up කරගත්තට පස්සේ bluetooth keyboard එකක් mouse එකක් උනත් පාවිච්චි කරන්න පුලුවන්.
  • Tv එකක් හෝ monitor එකක්
    රාස්බෙරි පයිවල screen එකක් නැති නිසා monitor එකක් වෙනම set කරගන්න වෙන්වා.
    Screen එකේ built-in speakers තියෙනවානම් රාස්බෙරි පයි එකට sounds play කරන්නත් පුලුවන්.
  • Monitor එකක් keyboard එකක් නැති නම් Headless විදිහට set up කරගන්නත් පුලුවන්.
  • HDMI
    ඔයගෙ රාස්බෙරි පයි එකේ HDMI output port එක්කුත් තියෙනවා. අලුත් tv monitor ගොඩාක් එක්ක ඒක වැඩ කරනවා.
    සමහර monitorවල DVI හෝ VGA ports තියෙන්නත් පුලුවන.(ඒව ගැනත් කියනවා)
    රාස්බෙරි පයි 4 එකේ micro HDMI port දෙකක් තියෙනවා. ඒනිසා monitor දෙකක් ගහන්නත් පුලුවන්.
    ඔයාට micro HDMI to HDMI කේබල් එකක් හෝ standard HDMI to HDMI කේබල් එකක් සහ micro HDMI to HDMI adapter එකක් ඕන වෙනව රාස්බෙරි පයි එක screen එකට සම්බන්ධ කරගන්න.
    රාස්බෙරි පයි 1,2 සහ 3වල තියෙන්නේ full-size HDMI port. එනිසා කෙලින්ම standard HDMI to HDMI කේබල් එක්කින් screen එක connect කරගන්න පුලුවන්.

රාස්බෙරි පයි 3 එක්ක HDMI port එක

 

  • DVI
    ඔයාගෙ screen එකේ DVI port එකක් නම් තියෙන්නෙ HDMI to DVI කේබල් එක්කින් connect කරන්න.
  • VGA
    සමහර screen වල VGA port තියෙන්නෙ. එතොකට ඔයාට පුලුවන් HDMI to VGA adapter එකක් පාවිච්චි කරන්න.

වැඩිපුර බඩු

  • කේස් (case) එකක්
    ඔයාට පුලුවන් ඕනිනම් case එකක් ගන්න ඔයාගෙ රාස්බෙරි පයි එකට. මේක අනිවාර්ය නෑ.. ඒත් රාස්බෙරි පයි එකේ ආරක්ෂාවට හොදයි.
  • Headphones හෝ speakers
    ලොකු රාස්බෙරි පයි model වල (රාස්බෙරි පයි zero වගේ ඒවා නැතුව) standard audio port එකක් තියෙනවා.
    කලින් කිවා වගේ screen එකේ built-in speakers තියේනම් ඒවගෙනුත් sound play කරන්න පුලුවන්.
  • Ethernet කේබල් එකක්
    ලොකු රාස්බෙරි පයි modelවල standard ethernet port එක්කුත් තියෙනවා internet එකට connect වෙන්න.
    රාස්බෙරි පයි zero වගේ එක්කට USB to Ethernet adapter එකක් පාවිච්චි කරන්න පුලුවන්.

රාස්බෙරි පයි 4, රාස්බෙරි පයි 3 සහ රාස්බෙරි පයි zero wifi වලිනුත් internet එකට connect කරන්න පුලුවන්
