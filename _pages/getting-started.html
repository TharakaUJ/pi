Raspberry pi වලට අලුත් ඔයාට දැවෙන ප්‍රශ්න ගොඩක් ඇති දැනටත්. අපේ මේ ලිපිය කියවල ඉවර උනාම ඒ ප්‍රශ්නවලට උත්තර හම්බවේවි.

අපි මුලින්ම ඔයා ගන්න ඕන රාස්බෙරි පයි එක මොකක්ද කියල බලලා, ඊලඟට ඒක setup කරගන්නෙ කොහොමද කියල මේ tutorial එකෙන් කියල දෙනවා.

ඔයා තාම රාස්බෙරි පයිවලට අලුත් නිසා පාවිච්චි කරන්න ලේසිම විදිහට අපි රාස්බෙරි පයි එක setup කරගමු.

ඉස්සරහට monitor එකක්වත් keyboard එකක්වත් නැතුව set up කරගන්න විදිහත් කියල දෙන්නම්කෝ..

රාස්බෙරි පයිවල තියෙන තවත් විශේෂත්වයක් තමා experiments කරන්න තියෙන හැකියාව. ඔයාගෙ වටින computer එක laptop එකෙන් අලුත් දෙයක් try කරනවට වඩා ගොඩක් ආරක්ෂිතව රාස්බෙරි පයි එකෙන් ඔයාට අලුත් දෙයක් කරන්න පුලුවන්. මොකද මේකෙ OS එක ගහන්නත් ලේසියි ඒවගේම පාවිච්චි කරන්න බැරිම වෙන්න මොනව හරි උනත් ලොකු ගාණක් අපතේ යන්නෙත් නෑනෙහ්.

රාස්බෙරි පයි වර්ග ගොඩක් තියෙනවා. ඒත් ගොඩක් අය තෝරගන්නෙ රාස්බෙරි පයි 4 model B. ඒක තමා අලුත්ම, වේගවත්ම හා වඩාත් පහසුවෙන්ම භාවිතා කරන්න පුලුවන් රාස්බෙරි පයි එක.එකෙත් ram එක 2GB එක ගොඩක් වැඩ කරගන්න ඇති. මේ තියෙන්නෙ රාස්බෙරි පයි 4එකක්. image reference: ranspberrypi.org මීට වඩා පොඩි රාස්බෙරි පයි වර්ගත් තියෙනවා. රාස්බෙරි පයි zero කියන්නේ එවගෙ එකක්. ඒවට current එක යනව අඩු නිසා රොබෝල හදන්න වගේ ගැලපෙනවා. ඒත් රාස්බෙරි පයි 4 එක පාවිච්චි කරන්න ලේසි නිසා ඒකෙන් මුලින් prototype එකක් හදල පස්සේ ඔන්නම් පොඩි එක්කට මාරු වෙන එක ලේසියි.
