Scratch කියන්නේ ඕනම කෙනෙක්ට (විශේෂයෙන් පොඩ්ඩන්ට) ඉතාමත් ඉක්මනින් ඉගෙන ගන්න programming language එකක්. programming ගැන මෙලෝ හසරක් ඔයා දන්නෙ නැත්තම් මෙතන පටන් ගන්න හොද තැනක්.
\nරාස්බෙරි පයි සහ Scratch අතර තවත් විශේෂ කාරණාවක් තමා Scratchවලින් රාස්බෙරි පයිවල තියෙන GPIO pin control කරන්නත් තියෙන හැකියාව. ඒ නිසා රාස්බෙරි පයි එකක් තියෙන ඔයාට ලේසියෙන්ම DIY projects කරගන්න පුලුවන්.
\nඅනික Scratchවලින් පොඩි පොඩි games හදාගන්නත් පුලුවන්.
\nඑහෙනම් බලමු කොහොමද Scratch install කරගන්නෙ කියල.ඔයාගේ රාස්බෙරි පයි එකේ desktop එකේ තියෙන menu එක open කරගන්න.ඊලගට preferences කියන එක click කරල ඒකෙ ඇතුලෙ තියෙන Recommended software කියන එක් ක්ලික් කරන්න.
\nඑතොකට මෙහෙම එකක් open වෙයි.
\nඊලගට පොඩ්ඩක් scroll කරල Scratch3 කියන software එක ඉස්සරහින් හරියක් දාන්න.(Scratch3 කියන්නෙ අලුත් version එකක් නිසා.) ඊලගට යට තියෙන Apply කියන button එක ඔබන්න. දැන් Scratch install වෙයි.

\n

මේ කරපු දේම terminal එකෙනුත් ඕන්නම් කරන්න පුලිවන්.
\nsudo apt install scratch
\nකියන command එක terminal එකේ type කරල enter කරන්න.

install වෙලා ඉවර උනාට පස්සේ menu එක open කරාම programming යටතේ ඔයාට Scratch3 බලාගන්න පුලුවන් වෙයි. ඒක ක්ලික් කරල Scratch open වෙයි.
\nඑහෙනම් ඊලග tutorial එකට කලින් පොඩ්ඩක් අත්හදා බලන්නකෝ.

Scratch කියන්නේ ඕනම කෙනෙක්ට (විශේෂයෙන් පොඩ්ඩන්ට) ඉතාමත් ඉක්මනින් ඉගෙන ගන්න programming language එකක්. programming ගැන මෙලෝ හසරක් ඔයා දන්නෙ නැත්තම් මෙතන පටන් ගන්න හොද තැනක්.

රාස්බෙරි පයි සහ Scratch අතර තවත් විශේෂ කාරණාවක් තමා Scratchවලින් රාස්බෙරි පයිවල තියෙන GPIO pin control කරන්නත් තියෙන හැකියාව. ඒ නිසා රාස්බෙරි පයි එකක් තියෙන ඔයාට ලේසියෙන්ම DIY projects කරගන්න පුලුවන්.

අනික Scratchවලින් පොඩි පොඩි games හදාගන්නත් පුලුවන්.

එහෙනම් බලමු කොහොමද Scratch install කරගන්නෙ කියල.
ඔයාගේ රාස්බෙරි පයි එකේ desktop එකේ තියෙන menu එක open කරගන්න.
ඊලගට preferences කියන එක click කරල ඒකෙ ඇතුලෙ තියෙන Recommended software කියන එක් ක්ලික් කරන්න.

එතොකට මෙහෙම එකක් open වෙයි.

ඊලගට පොඩ්ඩක් scroll කරල Scratch3 කියන software එක ඉස්සරහින් හරියක් දාන්න.(Scratch3 කියන්නෙ අලුත් version එකක් නිසා.) ඊලගට යට තියෙන Apply කියන button එක ඔබන්න. දැන් Scratch install වෙයි.

මේ කරපු දේම terminal එකෙනුත් ඕන්නම් කරන්න පුලිවන්.

sudo apt install scratch
