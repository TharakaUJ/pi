 අපේ පළවෙනි ghostව අල්ලල ඌව quarantine කරමු.
\nදැන් අපි කරන්න යන්නෙ අපි අල්ලන ghostලව හිර කරන් තියාගන්න quarantine folder එකක් හදාගන්න එක.
\nඇත්ත quarantine folder සාමාන්‍යයෙන් antivirus software එකෙන් තමා හැදෙන්නේ. දැනට අපි මේ විදිහට quarantine folder එකක් simulate කරමු.
\nඑහෙනම් Folder එකක් හදමු. shell prompt එකට එහා පැත්තෙන් මේක type කරල Enter කරන්න.

\n

mkdir quarantine/

\n

mkdir කියන්නෙ make directory කියන එක කොට කරල.
\nදැන් අලුත් directory එකක් හැදිල ඇති. ඔයාට ඕන්නම් ls කරල බලන්න පුලුවන්.
\nඔයාගෙ පලවෙනි ghostව අල්ලගමු.
\nකලින් ඉගෙන ගත්ත commands වලින් Documents directory එකට යන්න.
\nDocuments directory එකේ තියෙන දේවල් list කරන්න.

\n

මොනවහරි අමුතු දෙයක් හම්බුනාද?
\nfiles copy කරමු.
\nහම්බුන ghostව quarantine folder එකට copy කරගන්න මේක type කරන්න.
\ncp sp00ky.txt ~/quarantine/

\n

cp කියන්නේ copy කියන්න. ඊට පස්සෙ තියෙන්නෙ copy කරගන්න ඕන එක. ඊලගට ඔයාට copy කරගන්න ඕන තැන.
\nfiles remove කිරීම.
\nදැන් ghostව remove කරමු.
\nඒකට පහල තියෙන එක type කරන්න.
\nrm sp00ky.txt

\n

rm කියන්නෙ remove කියන්න.

\n

Note: remove කරනකොට පරිස්සම් වෙන්න ඕන මොකද එක පාරක් remove කරාම ආයෙ ගන්න බැරි වෙන්නම ඒක නැති වෙනවා.

\n
