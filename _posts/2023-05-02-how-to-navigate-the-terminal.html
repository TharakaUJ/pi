දැන් අපි කියල දෙන්න යන්නේ terminal එහෙට්ට මෙහෙට්ට navigate වෙන්නෙ කොහොමද කියල හරියට Pac-Man එයාගේ maze එකේ එහෙට මෙහෙට යනව වගේ.

Directory එක්ක තියෙන්නෙ මොනවද බලමු.

මේකට ඕන වෙන්නෙ ls command එක
Shell prompt එකට එහායින් ls කියල type කරල enter කරන්න.
ls කියන්නෙ list කියන එකට. command එකෙන් වෙන්නෙ මොකක්ද කියල දැන් තේරෙනව නෙහ්.

ඔයාට මේ වගේ එකක් බලගන්න පුලුවන් වෙයි. මේ තියෙන්නේ අපි දැන් ඉන්න directory එකේ තියෙන දේවල්.

කොහොමද අපි දැන් ඉන්න directory එක හොයාගන්නේ

ඒකට තියෙන්නෙ pwd කියල Type කරන්න. pwd කියන්නේ print working directory කියන එක කොට කරල. Type කරල Enter කරල බලන්න.
ඔයාට මේ වගේ එකක් බලාගන්න පුලුවන් වෙයි.

මේකේ තියෙන විදිහට දැන් අපි ඉන්නෙ home කියන directory එක ඇතුලෙ තියෙන username directory එක ඇතුලේ.
/ මේක තියෙන්නෙ ඔයාට directory වෙන් කරල පෙන්නන්න.
/home/username තමා ඔයාගෙ home directory එක මේක ඇතුලේ තමා ඔයාගෙ දේවල් save කරන්න තියෙන්නෙ. ඒ නිසා මේකට shortcut එක්කුත් තියෙනවා. ඒ තමා ~ සලකුණ.

කොහොමද directory මාරු කරන්නේ.

මේකට ඕන්න වෙන්නෙ cd command එක.
cd කියන්නේ change directory කියන එක කොට කරල.
ඔයාට කරන්න තියෙන්නෙ cd කියල type කරල හිස්තැනක් තියල ඔයාට යන්න ඕන directory එකේ නම type කරන එක.
හැබැයි එහෙම නම ගහල යන්න පුලුවන් වෙන්නෙ ඔයා ඉන්න directory එක අතුලේ ඔයාට යන්න ඕන directory එක තිබ්බොත් විතරයි. ඒ කියන්නෙ ls ගහපුවහම යන්න ඕන තැන පේන්න තිබ්බොත් විතරයි.
නැත්තම් ඔයාට සම්පූර්ණ path එකම type කරන්න වෙනවා. මෙන්න මේ වගේ.

හැබයි ඒ කොහොම උනත් ඔයා cd ~ කියල type කරොත් ඔයා ඔයාගෙ home directory එකට අනිවාරෙන් යනවා.

directory එක්කින් පස්සට යන්නත් ලේසි විදිහක් තියේ. ඒ cd වලින් පස්සේ .. දෙකක් type කරල.
.. කියන්නේ දැන් ඉන්න directory එකෙන් එකක් පස්සට ගියාම තියෙන directory එක. එතොකට cd .. කියන්නේ ආපස්සට යන්න කියන එක.

දැන් ඔයා පලවෙනි ghostව අල්ලන්න සූදානම්.
