රාස්බෙරි පයි එක start up කරමු.

\n\t\t\t\t
\n\t\t\t\t\n\t\t\t
\n\t\t\t
\n\t\t\t\t\n\t\t\t\t\n\t\t\t\t\n\t\t\t\t\n\t\t\t
\n\t\t\t\t

\n\t\t\t\t\n\t\t\t\t\n\t\t\t\t\n\t\t\t\t\n\t\t\t\t
ඔයාගෙ රාස්බෙරි පයි එකට වෙනම power switch එකක් නෑ. Current එකට ගහපු ගමන් ඒක on වෙනවා.Power supply එක plug එක්කට ගහල රාස්බෙරි පයි එකේ power port එකට connect කරන්න.
\nඔයාට රතු පාට LED එකක් පත්තු වෙනවා පෙනෙයි. දැන් අපේ රාස්බෙරි පයි එක boot වෙන ගමන් තියෙන්නේ.

\n

දැන් ඔයාගේ screen එකෙ උඩ වම් පත්තේ රාස්බෙරි ටිකක් පෙනෙයි.

\n

තව ටික්කින් රාස්බෙරි පයි dekstop එක ඔයාට පෙන්න ගනියි.
\n ඔයා පලවෙනි පාරට රාස්බෙරි පයි එකක් start up කරද්දි “Welcome to Raspberry Pi ” කියල application එකක් pop up වෙලා මුල් setup එක ගැන පොඩි guide එකක් දෙයි.
\nඒකේ කියන විදිහට next දීල country, language, timezone එහෙම set කරගන්න.
\nඊලඟට ඔයාගෙ username එකයි password එකයි set කරගන්න.

\n

Desktop එක monitor එක පුරාම තියෙන විදිහට ඔයාගෙ screen එක setup කරගන්න.

\n

Wifi connect කරගන්න.(ඔයාගෙ රාස්බෙරි පයි model එකේ wifi නැත්තම් ඔයාට මේක පේන එකක් නෑ.)
\nWifi connect වෙලා හරි වෙලාව පෙන්නනකන් ඉදල software update එකට check කරන්න කැමති නම් next දෙන්න ඊලඟට. නැත්තම් skip කරන්න.
\nඅන්තිමට Restart කියන එක ඔබන්න.Restart උනාට පස්සේ set up කිරිම ඉවරයි.
\nප්‍රශ්නයක් තිබ්බොත් comment එකක් දාන්න අමතක කරන්න එපා.
\n

ඔයාගෙ රාස්බෙරි පයි එකට වෙනම power switch එකක් නෑ. Current එකට ගහපු ගමන් ඒක on වෙනවා.
Power supply එක plug එක්කට ගහල රාස්බෙරි පයි එකේ power port එකට connect කරන්න.

ඔයාට රතු පාට LED එකක් පත්තු වෙනවා පෙනෙයි. දැන් අපේ රාස්බෙරි පයි එක boot වෙන ගමන් තියෙන්නේ.

දැන් ඔයාගේ screen එකෙ උඩ වම් පත්තේ රාස්බෙරි ටිකක් පෙනෙයි.

තව ටික්කින් රාස්බෙරි පයි dekstop එක ඔයාට පෙන්න ගනියි.

ඔයා පලවෙනි පාරට රාස්බෙරි පයි එකක් start up කරද්දි "Welcome to Raspberry Pi " කියල application එකක් pop up වෙලා මුල් setup එක ගැන පොඩි guide එකක් දෙයි.

ඒකේ කියන විදිහට next දීල country, language, timezone එහෙම set කරගන්න.

ඊලඟට ඔයාගෙ username එකයි password එකයි set කරගන්න.

Desktop එක monitor එක පුරාම තියෙන විදිහට ඔයාගෙ screen එක setup කරගන්න.

Wifi connect කරගන්න.
(ඔයාගෙ රාස්බෙරි පයි model එකේ wifi නැත්තම් ඔයාට මේක පේන එකක් නෑ.)

Wifi connect වෙලා හරි වෙලාව පෙන්නනකන් ඉදල software update එකට check කරන්න කැමති නම් next දෙන්න ඊලඟට. නැත්තම් skip කරන්න.

අන්තිමට Restart කියන එක ඔබන්න.Restart උනාට පස්සේ set up කිරිම ඉවරයි.
