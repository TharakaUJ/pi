මේක පටන් ගන්න මුලින්ම කරන්න තියෙන්නෙ Pac-Man ghost ලා ටික නිදහස් කරන file එක download කරගන්න.

Note: මේ file එකේ එහෙම ඇත්ත virus නෑ. මේක raspberry pi official website එකෙන් මේ game එකටම හදපු file එකක්.

Terminal එකේ ඔයාට ලේසියෙන්ම files download කරල ඒවා open කරන්න පුලුවන් bash පාවිච්චි කරල. Bash කියන්නෙ ඔයාගෙ command එක අහල කියන දේ කරන programme එකක්. Hacker කෙනෙක් වෙන්න මේ command පුලුවන් වෙන එක සෑහෙන වටිනවා.

Terminal එක open කරන්න. ඔයාට $ සලකුණක් පේන්න තියෙයි. ඒතමා shell prompt එක. ඒක පේනවා කියන්නේ terminal එක command එකක් ගන්න ලෑස්තියි කියල.

$ ට එහා පැත්තෙන් පහල තියෙන command එක type කරන්න.(නැත්තම් copy කරල paste කරන්න.)

Enter එක ඔබන්න. වචන ගොඩාක් terminal එකේ පෙනෙයි. ඒ download වෙනව සහ files එක execute වෙනවා කියන්න.

Note: අන්තිමට තියෙන | bash command එක download කරපු file එක එතෙනින්ම run කරන්න කියන්වා. සාමන්‍යයෙන් එහෙම කරන්න එපා කියනවා මොකද download උනේ මොකක්ද කියල බලන්න කලින් ඒක run වෙන නිසා. එතොකට virus එකක් open වෙන්න ඉඩකුත් තියෙනවානෙහ්.
ඒත් අපි මෙතන ඒක පාවිච්චි කරනවා ලේසියෙන්ම අපේ treasure hunt එක පටන් ගන්න.

Files එක run වෙලා ඉවර වෙලා command prompt එක ආයෙ පේන්න ගත්තම ctrl + l ඔබලා terminal එක clear කරගන්න. (නැත්තම් terminal එකේ clear කියල type කරල enter කරන්න)

දැන් තියෙන්නෙ maze එක පුරා ගිහින් ghostලා ඔක්කොම අල්ලගන්න.
